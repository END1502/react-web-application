## Getting Started

### Prerequisites

- NodeJS v16.18+

### Setup

Run `npm install` to download the dependencies.

### Running

To start the application, run the following command:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
